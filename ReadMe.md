# BRKH auto flow - Google Flow Automation

BRKH auto flow adalah extension Chrome khusus yang dikembangkan untuk 
meningkatkan efisiensi produksi aset microstock di BRKH Studio menggunakan 
teknologi Google Labs Flow.

## FITUR UTAMA
- Turbo Inject: Injeksi prompt langsung via React Fiber Bypass.
- Smart Queue: Antrean otomatis dengan indikator progress real-time.
- Quick Style Modifiers: Penambahan style visual instan ke prompt utama.
- Auto Download (Dewa Version): Mendownload hasil render baru (bukan kotak kosong) 
  melalui jalur Context Menu (Klik Kanan).
- Anti-Ban Logic: Jeda cooling-down 15 detik otomatis antar task.

## INSTALASI
1. Buka Chrome Browser dan masuk ke `chrome://extensions/`.
2. Aktifkan [Developer Mode] di pojok kanan atas.
3. Klik [Load Unpacked].
4. Pilih folder yang berisi file extension ini.
5. Pin icon BRKH auto flow di toolbar Chrome lu.

## TEKNOLOGI
- Manifest V3
- React Fiber Instance Injection
- Glassmorphism UI Aesthetic
